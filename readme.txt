A simple WordPress plugin that adds a homepage edit link to the admin sidebar

== Description ==

This plugin adds a link to the edit screen of your homepage. The link is added in the admin sidebar underneath the default 'Pages' menu item.

If you don't have a static homepage set, the link will not appear as there's no page to edit.
